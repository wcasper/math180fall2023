\question 
Harry and Terry are each told to calculate $8-(2+5)$. Harry gets the correct answer. Terry ignores the parentheses and calculates $8-2+5$. If Harry's answer is $H$ and Terry's answer is $T$, what is $H-T$?

\begin{oneparchoices}
\choice $-10$
\choice  $-6$
\choice 0
\choice 6
\choice 10
\end{oneparchoices}
\answerline
